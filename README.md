## 🚀 How to Run Any Project

All notebooks are built in **Google Colab** — no local installation needed.

1. Navigate into any project folder above
2. Open the `.ipynb` file
3. Click the **"Open in Colab"** button at the top
4. Upload the dataset CSV from the same folder (or run the GitHub raw URL line in the notebook)
5. Run all cells top to bottom — `Runtime → Run all`
